# "Артем на газели" — Спецификация проекта

## Executive Summary

Одностраничный лендинг для частного перевозчика "Артем на газели" в Тюмени. Цель — привлечение клиентов через интернет и автоматизация приёма заявок. Сайт должен произвести "вау-эффект" за счёт 3D-анимаций и передового дизайна уровня Awwwards.

## Problem Statement

Частный перевозчик не имеет онлайн-присутствия. Клиенты приходят только через "сарафанное радио". Сайт решает:
- Привлечение новых клиентов через поисковики и рекламу
- Мгновенный расчёт стоимости для клиента
- Автоматический приём заявок (без ручной обработки звонков)

## Success Criteria

- Клиент за 30 секунд понимает, что предлагается и сколько стоит
- Форма заявки отправляет уведомление на email/Telegram
- Сайт загружается < 3 секунд на мобильном 4G
- "Вау-эффект" от 3D-анимаций при сохранении производительности

## User Personas

### Частное лицо (60% трафика)
- Переезд квартиры/офиса, доставка мебели
- Ищет в Google/Яндекс "грузоперевозки Тюмень недорого"
- Хочет быстро узнать цену и оставить заявку
- Смартфон — основное устройство

### Малый бизнес (25% трафика)
- Регулярные доставки товаров, стройматериалы
- Ищет надёжного перевозчика с прозрачными тарифами
- Десктоп или планшет

### Маркетплейсы / стройки (15% трафика)
- Крупные или регулярные заказы
- Важна грузоподъёмность и габариты кузова

## Структура лендинга (секции)

### 1. Hero-секция
- Название "Артем на газели"
- Подзаголовок: "Грузоперевозки по Тюмени и межгороду"
- CTA-кнопка "Оставить заявку" / "Рассчитать стоимость"
- 3D-анимация газели (основной вау-элемент)

### 2. Услуги
- Квартирные переезды
- Офисные переезды
- Доставка стройматериалов
- Доставка для бизнеса
- Межгородные перевозки
- Карточки с иконками, glassmorphism-эффект

### 3. Преимущества
- Быстрая подача (от 30 мин)
- Честная цена — расчёт сразу
- Аккуратная перевозка
- Водитель-грузчик в одном лице

### 4. Тарифы
Стиль карточек как на референсе (светлый фон, тени, акцентные цвета для цен):

| Тариф | Условия |
|-------|---------|
| По городу | 900 ₽/час, каждый км свыше 10 км — 40 ₽/км |
| Пригород (до 50 км) | 900 ₽/час, +40 ₽/км |
| Межгород 50–100 км | Минималка 1.5 часа + 40 ₽/км |
| Межгород 100+ км | Минималка 2 часа (900 ₽/ч) + 40 ₽/км |
| Грузчик | 700 ₽/час (на лифте) |

### 5. Калькулятор стоимости
- Красивый UI с glassmorphism
- Поля: откуда, куда, тип груза, нужен ли грузчик
- Кнопка "Рассчитать" (логика — позже)
- Анимированный результат

### 6. О машине
- Газель тентованная
- Габариты кузова: 4.2 × 2 × 1.8 м
- Грузоподъёмность: до 1.5 тонн
- Плейсхолдеры для фото (2 ракурса)

### 7. Зона покрытия
- Тюмень, пригород, межгород в любом направлении
- Текстовое описание (без карты)

### 8. FAQ
- Как заказать?
- Какие грузы перевозите?
- Есть ли страховка груза?
- Сколько стоит подача?
- Можно ли перевозить в выходные?
- Как производится оплата?

### 9. Контакты + Форма заявки
- Форма: имя, телефон, откуда-куда, описание груза, дата
- Телефон (заглушка)
- Email (заглушка)
- Telegram @max (заглушка)

### 10. Плавающие элементы
- Кнопка Telegram (ссылка на @max)
- Кнопка Email
- Всегда видны, анимированные

## Design Specification

### Цветовая палитра (по референсу)
- **Фон**: #FFFFFF / #F8F9FC (секции чередуются)
- **Основной текст**: #1A1A2E
- **Primary (синий/индиго)**: #4F46E5
- **Accent (оранжевый, для цен)**: #F97316
- **Accent 2 (красный, для акцентов)**: #EF4444
- **Accent 3 (фиолетовый)**: #8B5CF6
- **Карточки**: белые с мягкой тенью, border-radius: 16px
- **Glassmorphism**: backdrop-filter: blur(20px), bg rgba(255,255,255,0.7)

### Типографика
- **Основной шрифт**: Unbounded (Google Fonts)
- **Вспомогательный**: Inter или system-ui для body-текста

### Стилистика
- Light Mode
- Glassmorphism для карточек и оверлеев
- Мягкие тени
- Плавные анимации при скролле
- 3D-элементы в hero и транзициях

### Адаптивность
- Mobile-first (основная аудитория — смартфоны)
- Бургер-меню на мобильных
- Десктоп: полная навигация в хедере

## Technical Architecture

### Stack
- **Framework**: Next.js 14 (App Router) + TypeScript
- **Styling**: Tailwind CSS
- **3D/Анимации**: Three.js + React Three Fiber + Framer Motion
- **Шрифт**: Unbounded (Google Fonts)
- **Деплой**: Vercel (бесплатный план для старта)

### Форма заявки
- Фронтенд-валидация
- Бэкенд: Next.js API route `/api/contact`
- Отправка: пока заглушка, позже — Telegram Bot API + email (nodemailer)

## Non-Functional Requirements

- **Performance**: LCP < 2.5s, CLS < 0.1, FID < 100ms
- **SEO**: Мета-теги, structured data для local business
- **Accessibility**: контраст, alt-тексты, keyboard navigation
- **Браузеры**: Chrome, Safari, Firefox (последние 2 версии)

## Out of Scope (v1)

- Яндекс.Карты
- Онлайн-оплата
- Мультиязычность
- Отзывы клиентов
